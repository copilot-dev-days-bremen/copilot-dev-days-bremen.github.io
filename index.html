Session

Beginner

Github Copilot Overview: What is this all about and what does it include?

his session provides a concise overview of GitHub Copilot, explaining what it is, how it works, and what capabilities it includes today. We’ll look at how Copilot supports developers across the software lifecycle—from writing and understanding code to testing and documentation—while clarifying where it adds real value and where human judgment remains essential. The goal is to demystify the tool, set realistic expectations, and give you a clear picture of how GitHub Copilot fits into modern development workflows.

Sven Sieverding

Session

Beginner

Spec Driven Development

Spec Driven Development turns specifications into the true starting point of software development. In this session, you’ll see how clear, well-structured specs—requirements, examples, and acceptance criteria—can be used directly by GitHub Copilot to generate code, tests, and refactorings. Learn how shifting from code-first to intent-first development leads to better results, fewer misunderstandings, and more predictable software—with Copilot as a powerful implementation partner rather than just a code autocomplete.

Jörn Syrbe

Venue

How to find us

Venue

Bremen, Schuppen 1

You can find us at

Schuppen 1
Konsul-Smidt-Straße 20
28217 Bremen

Just take the main entrance and walk straight ahead into the boulevard. Don't get distracted by all the classic cars. The session spaces are on the back left side of boulevard (Search for the signs mentioning "Crew Spaces").

If you use public transportation, you can take either use
  • the tram lines 3 or 5 to the "Konsul-Smidt-Straße" stop
  • bus line 26 to the "Überseetor" stop
and then take a 5-minute walk to the venue.

If you come by car, there are several parking options nearby, including the "BREPARKplatz Überseestadt" and street parking in the area.
We recommend using public transportation if possible, as parking can be limited during the event.
